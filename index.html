How To Use It

A quick weaving draft walkthrough.

Start by sizing the draft, then build the structure, add color, and refine the layout with insert controls.

1

Set your draft size

Use the controls at the top to choose the number of shafts, treadles, warp ends, and picks. Changing these values in the current editor preserves existing data when possible.

2

Build the structure

Place each warp end on a shaft, then connect treadles to shafts in the tie-up. The tie-up stays structural and prints as solid black.

3

Add warp and weft color

Use the pattern colors row to add or edit colors, select one, then click threading and treadling cells to assign warp and pick colors.

4

Insert ends and picks without losing work

Use the small + and - controls above the threading grid and to the right of the treadling grid to insert or remove columns and rows while keeping the rest of the draft intact.

Draft Layout

Structure of a draft.

  • Threading: which shaft each warp end is threaded through, plus its warp color.
  • Tie-up: which shafts lift when a treadle is pressed.
  • Treadling: the order of treadles across picks, plus the weft color for that pick.
  • Drawdown: the woven result calculated from structure and color together.
