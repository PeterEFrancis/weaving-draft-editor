How To Use It

A quick weaving draft walkthrough.

The editor is split into threading, tie-up, treadling, and drawdown. You build the structure by clicking directly inside those grids.

1

Set your draft size

Use the controls at the top to choose the number of shafts, treadles, warp ends, and picks. Changing these values in the current editor preserves existing data when possible.

2

Build the threading and tie-up

Click one shaft for each warp end in the threading area, then click tie-up cells to connect treadles to shafts.

3

Enter the treadling

Click the treadling grid to assign treadles to each pick and build your sequence row by row.

4

Insert ends and picks without losing work

Use the small + and - controls above the threading grid and to the right of the treadling grid to insert or remove columns and rows while keeping the rest of the draft intact.

What the panels mean

  • Threading: which shaft each warp end is threaded through.
  • Tie-up: which shafts lift when a treadle is pressed.
  • Treadling: the order of treadles across picks.
  • Drawdown: the woven result calculated from the other three sections.

Useful note

The Reset button clears the draft completely, so it’s helpful when you want a fresh start. For small structural changes, use the insert and delete controls instead.
